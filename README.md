# testing-gulp-inject-empty

## Installation

Clone `testing-gulp-inject-empty` using git:

```bash
git clone https://github.com/joakimbeng/testing-gulp-inject-empty.git
```

Then install the dependencies with:

```bash
npm install
```

## Tests

To run the test which verifies that gulp-inject's `empty` option works in conjunction with empty streams run:

```bash
npm test
```
